# Looping & String Manipulation in Python

This repository contains practice questions and examples based on:
- Python loops (for, while)
- String operations and manipulation
- Basic logic building

## Contents
- `Looping and string manipulation.ipynb`  
  Jupyter Notebook with Python practice questions and examples related to loops and strings.

## Purpose
This project is created for:
- Practicing Python basics
- Improving logic building
- Understanding how loops work with strings

## Topics Covered
- For loop
- While loop
- String indexing and slicing
- String methods
- Pattern and basic problem solving

## How to Use
1. Download the notebook file.
2. Open it using Jupyter Notebook or Google Colab.
3. Run each cell and try modifying the code for practice.
